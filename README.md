# Wireshark Beginner Network Traffic Analysis

## What this project is
This project shows how I used Wireshark to capture and observe normal internet traffic on my computer.

## What I did
- Installed Wireshark on Windows
- Captured live network traffic
- Visited normal websites
- Used simple filters (DNS and HTTP) to reduce noise

## What I learned
- How to start and stop packet captures
- How to apply basic display filters
- How normal internet traffic looks
- Why filtering is useful for SOC analysts

## Tools used
- Wireshark
- Windows 10
